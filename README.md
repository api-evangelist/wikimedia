# Wikimedia

Wikimedia provides REST APIs for accessing Wikipedia, Wikidata, Commons, and other Wikimedia project content including page summaries, search, media files, page revision history, and usage metrics. The suite includes the MediaWiki Action API for bot and edit operations, the MediaWiki REST API for modern content access, and the Wikimedia Enterprise API for commercial-scale high-volume data reuse across 920+ datasets and 360+ languages.

## APIs

- **MediaWiki Action API** - Full wiki feature access via api.php including search, page operations, authentication, and bot automation
- **MediaWiki REST API** - Modern RESTful interface for reading and writing pages, revisions, media, and search
- **Wikimedia REST API (RESTBase)** - High-performance cached content access at /api/rest_v1/ including page summaries, HTML, and metrics
- **Wikimedia Enterprise API** - Commercial-grade on-demand, realtime, and snapshot APIs for high-volume content reuse
